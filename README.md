### 🌫️ Air Quality Index Analysis
Worked on an Exploratory Data Analysis (EDA) project using AQI data from Indian metropolitan cities collected from a Kaggle dataset covering 2020–2024. Since the dataset did not include direct AQI values or category ranges, referred to the official Indian AQI guidelines and formulas to calculate daily AQI levels for each city. Classified air quality into categories such as low, medium, high, and extreme based on pollutant concentrations. Analyzed the relationship between pollutants and AQI using correlation analysis to identify the pollutants that most affected air quality in different cities. Compared pollution patterns across metropolitan cities, ranked cities based on overall AQI performance, and visualized insights using Python libraries for data analysis and visualization..
